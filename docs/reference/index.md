# Reference

This reference is organized by type family instead of one long page.

## Type families

- [Characters](chars.md): `utf8_char` and `utf16_char`
- [String Views](string-views.md): borrowed validated UTF-8 and UTF-16 text
- [Owning Strings](owning-strings.md): mutable UTF-8 and UTF-16 text
- [Views, Literals, and Formatting](views-literals-formatting.md): adaptors, literal operators, formatting, streaming, and hashing

## Reading the reference

The docs focus on:

- semantics
- API categories
- indexing rules
- common usage patterns

For exact signatures, include paths, and implementation details, use the headers themselves together with this reference.
